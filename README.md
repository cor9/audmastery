# Audition Mastery: The Booking Program Landing Page

A conversion-optimized landing page for Child Actor 101's 12-week audition training program.

## Setup for GitHub Pages

1. **Add the hero image**: Place `auditionmastertmast.jpg` in the root directory of this repository.

2. **Enable GitHub Pages**:
   - Go to your repository settings
   - Navigate to "Pages" in the left sidebar
   - Under "Source", select the branch containing your files (usually `main` or `master`)
   - Select the root directory (`/`)
   - Click "Save"

3. **Access your site**: Your page will be available at:
   ```
   https://[your-username].github.io/[repository-name]/
   ```
   Or if using a custom domain:
   ```
   https://auditionmastery.childactor101.com
   ```
